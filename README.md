# Matlab-Figure

[![File Exchange Badge](https://www.mathworks.com/matlabcentral/images/matlab-file-exchange.svg)](https://de.mathworks.com/matlabcentral/fileexchange/183308-matlab-figure) [![GitHub top language](https://img.shields.io/github/languages/top/mathworks/climatedatastore)](https://matlab.mathworks.com/)

Miscellaneous tools to support graphical visualization in Matlab

## Using

### Arrange all open Figures in Grid

```matlab
figure_arrange.grid();
```

## Testing

Open some figures and arrange them in a grid:

```matlab
figure_arrange.example.grid();
```
